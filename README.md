linc_ffmpeg
=============

Lite hxcpp externs for [ffmpeg](https://github.com/FFmpeg/FFmpeg) which follow [linc](https://snowkit.github.io/linc/) conventions.

`haxelib git linc_ffmpeg https://github.com/Dimous/linc_ffmpeg.git`
